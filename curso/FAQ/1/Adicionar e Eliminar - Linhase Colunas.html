📊 Tabelas: Adicionar e Eliminar Linhas/Colunas Layout > Linhas e Colunas

Manipular a estrutura de uma tabela no Word é uma tarefa comum. Você pode inserir ou excluir linhas e colunas a qualquer momento, sem perder a formatação do conteúdo. Abaixo, um guia visual com todas as opções disponíveis na guia Layout (ferramentas de tabela).

⚡ No Word, as opções aparecem automaticamente na guia Layout (em "Ferramentas de Tabela") quando você clica dentro da tabela.

➕ Adicionar linhas e colunas

⬆️⬇️ Linhas
  1. Clique em qualquer célula da linha adjacente.
  2. Vá até a guia Layout (Ferramentas de tabela).
  3. No grupo Linhas e Colunas, clique em:
    • Inserir Acima (insere antes)
    • Inserir Abaixo (insere depois)
⬅️➡️ Colunas
  1. Clique em qualquer célula da coluna adjacente.
  2. Vá até a guia Layout (Ferramentas de tabela).
  3. No grupo Linhas e Colunas, clique em:
    • Inserir à Esquerda (insere antes)
    • Inserir à Direita (insere depois)
⌨️ Atalho rápido (linhas): Posicione o cursor no final de uma célula e pressione Tab para criar uma nova linha abaixo. Para colunas, infelizmente não há atalho direto, use o menu Layout.

➖ Eliminar linhas, colunas ou a tabela inteira

🗑️ Linha

Selecione a(s) linha(s) → Layout → Eliminar → Eliminar Linhas.

🗑️ Coluna

Selecione a(s) coluna(s) → Layout → Eliminar → Eliminar Colunas.

🗑️ Tabela inteira

Clique dentro da tabela → Layout → Eliminar → Eliminar Tabela.

⚠️ Cuidado com a tecla Delete: Pressionar Delete ou Backspace com a célula selecionada apenas limpa o conteúdo (texto/números), não elimina a estrutura (linha/coluna). Para remover a estrutura, use sempre o menu Eliminar na guia Layout ou o clique com botão direito → Eliminar Células → escolha a opção correta.

📌 Dicas de produtividade

Ação Como fazer no Word Observação
Inserir linha abaixo Tab na última célula da linha Funciona apenas no final da tabela
Inserir linha acima Clique direito → Inserir → Inserir Linhas Acima Ou use o menu Layout
Selecionar linha inteira Clique na margem esquerda da linha (cursor vira seta) Facilita para eliminar ou copiar
Selecionar coluna inteira Clique no topo da coluna (cursor vira seta preta) Funciona como no Excel
Eliminar múltiplas linhas Selecione várias linhas → Layout → Eliminar Linhas Selecione com Shift ou arrastando
💡 Dica bônus: Se você precisa mesclar células ou dividir uma célula, essas opções também ficam na guia Layout (grupo Mesclar). Use para criar cabeçalhos personalizados ou layouts complexos.

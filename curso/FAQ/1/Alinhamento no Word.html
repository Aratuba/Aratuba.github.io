📐 Alinhamento no Word Imagens · Figuras · Caixas de Texto

Tutorial completo com exemplos visuais — tudo o que você precisa saber para posicionar elementos perfeitamente nos seus documentos.

No Microsoft Word, o alinhamento é uma ferramenta essencial para criar documentos organizados e profissionais. Este tutorial cobre as três principais categorias de elementos que você vai alinhar no dia a dia: imagens, figuras (com legendas) e caixas de texto.

🖼️ Alinhamento de Imagens 3 opções principais

No Word, você pode alinhar imagens de duas formas: em linha com o texto (como se fosse um caractere) ou com quebra de texto (flutuante). Aqui focamos nos alinhamentos mais usados: esquerda, centralizado e direita.

⬅️ Alinhamento à esquerda wrap: Quadrado
A

Imagem alinhada à esquerda

O texto flui ao redor da imagem pelo lado direito. Ideal para documentos com muitas imagens e texto descritivo.

🔹 Use para: relatórios, artigos, posts.
Word: Inserir → Imagem → Formatar → Alinhar à Esquerda
⬆️ Alinhamento centralizado wrap: Superior e inferior
B

Imagem centralizada

A imagem fica no centro da linha ou da página. O texto pode ficar acima e abaixo (wrap Superior/Inferior) ou ao redor.

🔹 Use para: títulos, destaques, capas.
Word: Inserir → Imagem → Formatar → Centralizar
➡️ Alinhamento à direita wrap: Quadrado

Imagem alinhada à direita

O texto flui ao redor da imagem pelo lado esquerdo. Muito usado para notas, sidebars e elementos decorativos.

🔹 Use para: chamadas laterais, assinaturas.
C
Word: Inserir → Imagem → Formatar → Alinhar à Direita

📌 Opções de quebra de texto (wrap)

No Word, além do alinhamento, você pode definir como o texto se comporta em relação à imagem. Estas são as principais:

# Opção Descrição Quando usar
🔲 Em linha A imagem se comporta como um caractere grande. Alinha com o texto. Documentos simples, formulários.
▢ Quadrado O texto contorna a imagem em formato retangular. Maioria dos casos, boa legibilidade.
⊡ Compacto O texto contorna bem próximo à imagem, seguindo o contorno. Imagens com formas irregulares.
⊹ Atravessar O texto preenche o espaço ao redor, inclusive vazios da imagem. Efeitos visuais criativos.
⬆⬇ Superior e inferior O texto fica apenas acima e abaixo da imagem (não nas laterais). Destaques, imagens grandes.
🔙 Por trás do texto A imagem fica como marca d'água, o texto passa por cima. Planos de fundo, marcas d'água.
🔜 À frente do texto A imagem sobrepõe o texto. Stickers, anotações, rótulos.
💡 Dica: Para acessar as opções de wrap, clique com o botão direito sobre a imagem → Quebra de Texto → escolha a opção desejada.

🏷️ Alinhamento de Figuras (com legenda) Figura + legenda

Uma figura no Word é uma imagem (foto, gráfico, diagrama) que possui uma legenda descritiva. O alinhamento da figura pode ser feito flutuante (com wrap) ou inline.

📋 Guia prático — 5 passos para alinhar qualquer elemento

  1. Selecione o elemento — clique na imagem, figura ou caixa de texto que deseja alinhar.
  2. Abra a guia "Formatar" — no menu superior (ela aparece quando o elemento está selecionado).
  3. Escolha o tipo de wrap — no grupo Organizar, clique em Quebra de Texto e selecione a opção desejada (Quadrado, Compacto, etc.).
  4. Alinhe o elemento — no mesmo grupo Organizar, use os botões Alinhar (Esquerda, Centralizar, Direita). Para caixas de texto, use também Distribuir.
  5. Ajuste a posição fina — use as setas do teclado ou arraste com o mouse. Para precisão, use Posição no grupo Organizar.
🚀 Atalho rápido: Selecione o elemento e use Ctrl + E (centralizar), Ctrl + L (esquerda) ou Ctrl + R (direita) — mas isso alinha o conteúdo interno. Para o posicionamento da caixa em si, use os botões da guia Formatar.

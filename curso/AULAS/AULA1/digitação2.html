⌨️ O que é um teclado?

O teclado é um dispositivo de entrada utilizado para digitar textos, números e comandos no computador.

O modelo mais utilizado no Brasil é o ABNT2, criado conforme as normas da Associação Brasileira de Normas Técnicas (ABNT).

Sua principal característica é possuir:

  • tecla Ç
  • tecla Alt Gr
  • teclas para acentuação em português

🧩 Principais partes do teclado

O teclado é dividido em cinco áreas:

  • Teclas de digitação – letras, números e símbolos
  • Teclas de função – F1 a F12
  • Teclado numérico – números e operadores
  • Teclas de navegação – setas, Home, End, etc.
  • Teclas de controle – Ctrl, Alt, Shift, etc.

🔑 Teclas principais

1. Teclas de Digitação

São utilizadas para escrever textos. Exemplos: A B C D E · 1 2 3 4 5 · Ç

Servem para digitar letras, números, símbolos e pontuação.

2. Enter Enter

Funções:

  • confirma uma ação
  • cria uma nova linha em um texto
  • executa comandos
Exemplo: Ao terminar de escrever um parágrafo, pressione Enter para ir para a próxima linha.
3. Espaço (Space) Espaço

É a maior tecla do teclado. Serve para separar palavras.

Exemplo: Bom dia (com espaço) vs Bomdia (sem espaço).
4. Backspace ← Backspace

Apaga o caractere localizado à esquerda do cursor.

Exemplo: Casaa| → pressionando Backspace → Casa|
5. Delete (Del) Delete

Apaga o caractere localizado à direita do cursor. Também pode excluir arquivos quando selecionados.

6. Shift Shift

Serve para:

  • escrever letras maiúsculas
  • acessar o símbolo superior das teclas
Exemplos: 1 → ! · 2 → @ · a → A
7. Caps Lock Caps Lock

Mantém todas as letras em maiúsculo.

Exemplo: bom dia → com Caps Lock ativado → BOM DIA
8. Tab Tab ↹

Serve para:

  • criar recuos
  • mudar de campo em formulários
  • organizar tabelas
9. Esc (Escape) Esc

Cancela ou fecha algumas operações. Exemplos: fechar menus, sair da tela cheia, cancelar uma ação.

10. Ctrl (Control) Ctrl

É utilizada junto com outras teclas. Atalhos mais usados:

11. Alt Alt

Também funciona em conjunto com outras teclas.

Exemplo: Alt + Tab alterna entre programas abertos.
12. Alt Gr Alt Gr

Exclusivo do teclado brasileiro. Permite digitar símbolos especiais.

Exemplos: @ £ ¢ € ² ³
13. Tecla Windows ⊞ Windows

Abre o Menu Iniciar.

Atalhos úteis:

  • Windows + E – Abre o Explorador de Arquivos
  • Windows + L – Bloqueia o computador
  • Windows + D – Mostra a Área de Trabalho
  • Windows + R – Executar

15. Setas ↑ ↓ ← →

Servem para mover o cursor. Também navegam entre menus e documentos.

16. Home Home

Move o cursor para o início da linha.

17. End End

Move o cursor para o final da linha.

18. Page Up Page Up

Sobe uma página.

19. Page Down Page Down

Desce uma página.

20. Insert Insert

Alterna entre inserir texto e sobrescrever texto. Hoje é pouco utilizada.

21. Print Screen (PrtSc) PrtSc

Captura a tela. No Windows, também pode ser usada com Windows + Shift + S para selecionar apenas uma parte da tela.

22. Num Lock Num Lock

Liga ou desliga o teclado numérico. Quando ativado, as teclas 7 8 9 / 4 5 6 / 1 2 3 / 0 funcionam como números.

23. Scroll Lock Scroll Lock

Pouco utilizada atualmente. Em alguns programas altera a forma de navegação.

24. Pause/Break Pause

Quase não é utilizada atualmente. Antigamente interrompia alguns processos.

O teclado brasileiro usa teclas mortas para os acentos. Primeiro pressiona-se o acento e depois a letra.
